# 计划(Planner) - 已废弃

> **注意：** 自新版本起，Planner计划组件已被废弃。推荐使用**智能体模版（Agent Template）**来替代原有的Planner功能。

## 背景

在早期版本中，Planner影响实际Agent工作时的协作与执行策略，agentUniverse可以视为一个Pattern Factory，Planners是pattern的协作与执行思想的实际载体。

## 迁移指南

现在，原有的Planner功能已由智能体模版（Agent Template）替代。智能体模版提供了更灵活、更强大的协作与执行策略定义方式。

请参考以下文档了解智能体模版：
- [沉淀与使用智能体模版](../../../开始使用/5.沉淀与使用智能体模版.md)
- [智能体模版](../智能体/智能体模版.md)

### 对应关系
